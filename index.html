Open Finder Folders in Any App 在 Finder 中一键打开任意应用

Add a toolbar button or right-click menu to Finder, and instantly open any folder in Terminal, VS Code, iTerm2, or any app you choose. 在 Finder 工具栏添加按钮或使用右键菜单，即可快速用终端、VS Code、iTerm2 或任何应用打开文件夹。

Features 功能特点

🗂

Toolbar Button 工具栏按钮

A dropdown menu right in the Finder toolbar. One click to open the current folder in any app. 在 Finder 工具栏添加下拉菜单按钮，一键用任意应用打开当前文件夹。

📋

Context Menu 右键菜单

Right-click on any folder and find OpenWith in the context menu. Works everywhere in Finder. 右键点击任意文件夹即可在菜单中找到 OpenWith，在 Finder 任何位置使用。

⚡️

Smart Commands 智能命令

Auto-detects popular dev tools and generates optimal launch commands. Supports custom commands with %PATH%. 自动识别常用开发工具并生成最佳启动命令，支持通过 %PATH% 自定义命令。

📎

Copy Path 复制路径

Quickly copy the current directory path to clipboard from the toolbar menu. 从工具栏菜单快速复制当前目录路径到剪贴板。

🌐

21 Languages 21 种语言

Full localization in 21 languages including English, Chinese, Japanese, Korean, French, German, and more. 完整支持 21 种语言，包括中文、英文、日文、韩文、法文、德文等。

🪶

Lightweight 轻量无后台

No background processes, no network requests, no tracking. Just a simple, fast Finder extension. 无后台进程、无网络请求、无数据追踪。简单、快速的 Finder 扩展。

Context Menu 右键菜单

Right-click any folder to find OpenWith in the context menu. All your configured apps are just a click away. 右键点击任意文件夹，在菜单中找到 OpenWith 子菜单，所有已配置的应用触手可及。

Easily add new apps by selecting the .app file. OpenWith auto-detects the app name, icon, and generates the best launch command. 选择 .app 文件即可轻松添加新应用。OpenWith 自动识别应用名称、图标，并生成最佳启动命令。

Settings 设置

Configure language preferences, learn how to enable the extension, and customize your experience. 配置语言偏好，了解如何启用扩展，自定义你的使用体验。

Quick Setup 快速设置

1

Add Your Apps 添加应用

Open OpenWith and add the apps you want to use (Terminal, VS Code, iTerm2, etc.). 打开 OpenWith，添加你常用的应用（终端、VS Code、iTerm2 等）。

2

Enable Extension 启用扩展

Go to System Settings → Privacy & Security → Extensions → Added Extensions, and enable OpenWith. 前往系统设置 → 隐私与安全性 → 扩展 → 添加的扩展，启用 OpenWith。

3

Add to Toolbar 添加到工具栏

In Finder, right-click the toolbar → Customize Toolbar, then drag the OpenWith button to the toolbar. 在 Finder 中右键工具栏 → 自定义工具栏，将 OpenWith 按钮拖入工具栏。

4

Start Using 开始使用

Click the toolbar button or right-click a folder to open it in your favorite app! 点击工具栏按钮或右键文件夹，即可用你喜欢的应用打开！
